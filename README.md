# Automated-Email-System

An automated customer feedback and churn-risk detection system.

This project sends monthly feedback requests, collects structured responses, calculates a churn risk score, and flags high-risk customers before they leave.

📅 Monthly automated feedback emails (cron-based)

📝 Feedback submission endpoint

🧠 Rule-based churn risk scoring

🚨 High-risk detection logic

🗄 PostgreSQL data storage

🧱 Clean backend architecture (routes → controllers → services → utils)

🛠 Tech Stack

Node.js

Express

PostgreSQL

node-cron

SendGrid (email provider)
